1. Введение

Boolean (Булев или логический тип данных) — примитивный тип данных в информатике, который может принимать только два возможных значения, иногда называемых истиной (true) и ложью (false). True и false также называются булевыми (логическими) литералами*.

*Литерал — запись, представляющая собой фиксированное значение и использующаяся в JS для представления значений. Читайте подробнее о литералах в документации.

Булевый литерал не содержит кавычек (так как с кавычками это была бы уже строка).

В JS true или false могут означать и другие значения, благодаря приведению типов.

Преобразование к логическому типу происходит в логическом контексте или вызывается логическими операторами (&&, ||, !). Такое преобразование называется неявным.

Возвращают false Возвращают true
0, -0, NaN, пустая строка в любых кавычках, null, undefined, false Все остальные значения, включая отрицательные числа, пустой массив или объект.

Явное преобразование происходит с использованием вызова функции Boolean:

Boolean([123]);

true

Boolean('');

false
