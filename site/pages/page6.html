1. Циклы

Представим, вы выбрали смартфон в магазине и пошли на кассу, где оказалась очередь. Продавцу необходимо поочередно обработать заявку каждого покупателя, то есть он выполняет одно и то же действие несколько раз, пока не кончится очередь. Для автоматизации множества одинаковых действий в программировании придумали циклы.

Циклом называется многократное выполнение одних и тех же действий. Например, чтобы программа сосчитала от 1, скажем, до миллиона, в ней нужно записать цикл. Цикл — это та часть кода программы, которая выполняется заданное количество раз. У цикла, как правило, есть условие выхода (когда он должен завершиться).

В JavaScript, как и в других языках программирования, есть три вида циклов:

➤ for

➤while

➤do...while

Каждый из них обособляется фигурными скобками. Пойдём по порядку.

1. Цикл for

Это самый популярный цикл.

Синтаксис:

for ([initialization]; [condition]; [incrementExpression]) {

Statement

}

Statement внутри тела цикла (всё, что заключено в фигурные скобки) выполняется, если выражение condition возвращает true.

IncrementExpression выполняется каждую итерацию.

Порядок действий:

// Инициализировали переменную i, которая является счётчиком
// Объявили условие выхода из цикла i < 10
// Написали выражение, которое будет менять шаг цикла (инкремент)

for (let i = 0; i < 10; i = i + 1) {

console.log(i)

}

Если условие ложно, то ничего не выводится.

Вот так можно создать бесконечный цикл, который никогда не завершится:

for (;;) {

console.log('infinity loop')

}

2. Цикл while

While — цикл с предусловием. Не такой популярный как for, но иногда используется.

Синтаксис:

while (condition) {

Statement

}

Statement выполняется до тех пор, пока выражение condition возвращает true.

let i = 0;

while (i < 10) {

i = i + 1
console.log(i)

}

Если условие ложно, ничего не выводится.

Бесконечный цикл:

while (true) {

console.log('infinity loop')

}

Цикл do...while

Используется редко. Этот цикл называется «циклом с постусловием». Особенность в том, что он выполнится по крайней мере один раз, даже если условие ложно.

let i = 0;

do {

i = i + 1
console.log(i)

} while (i < 10)

Выводятся числа от 1 до 10.

Бесконечный цикл:

do {

console.log('infinity loop')

} while (true)

Помимо вышеперечисленных, в JavaScript также есть и специфичные циклы: for..of и for..in. Их мы рассмотрим в дальнейших уроках.

Альтернатива циклам

Кроме этого, существует специальный метод массива .forEach(), который является мутирующим методом массива, а также метод .map(), который является немутирующим (его мы рассмотрели ранее).

Они могут заменить классические циклы при работе с массивами. Какой из них выбрать, зависит от задачи и ожидаемого результата.

Метод .forEach(cb)

Он принимает функцию как аргумент и ничего не возвращает. Если внутри cb изменять элементы, то исходный массив также будет изменён.

arr.forEach(function(item, index, array) {

// возвращается новое значение вместо элемента

});

Где:

item — очередной элемент массива,
index — его индекс,
array — сам массив

Задание

Впишите в поле результат, который выведет консоль на это действие:

let arr = [1,2,3]

arr.forEach(function(item, index, array) {

console.log(item)

});

Ответ

123

Задание 7.

Дан массив. Нужно вывести в консоль количество чётных и нечётных элементов в массиве. Если в массиве есть нулевой элемент, то он учитывается и выводится отдельно. При выполнении задания необходимо учесть, что массив может содержать не только числа, но и, например, знаки, null и так далее.
